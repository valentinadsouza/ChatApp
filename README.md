ChatApp

A simple real-time chat application built with PHP, JavaScript, HTML/CSS, and MySQL.


Features :
User registration and login system.
Real-time chat between users.
Show/hide password functionality.
Search for users to chat with.
Upload and display profile/chat images.

Technologies Used
Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL
Assets: Images stored in images/ folder
Setup Instructions

Clone/Extract Project:
Place the ChatApp folder inside your local server root (htdocs for XAMPP).
Database Configuration:
Create a MySQL database.
Update database credentials in php/config.php.
Run the Application:
Start Apache and MySQL servers.

Open http://localhost/ChatApp/index.php in your browser.

User Registration & Login:
Sign up a new user via signup.php.
Log in with your credentials.
